Course Description

Human–AI Collaboration & Workflow Automation

This course introduces students to the design and implementation of human–AI collaborative workflows and automation systems. Using no-code and low-code platforms, students will integrate large language models, multi-modal tools, and productivity applications to streamline real-world processes. Topics include task design, agent orchestration, retrieval-augmented generation (RAG), and human-in-the-loop evaluation. Emphasis is placed on ethical, reliable, and equitable automation for business, education, and community applications.

Tools you'll work with: Zapier · Make (Integromat) · Microsoft Power Automate · ChatGPT Agents · Claude Projects · RAG frameworks

What You'll Learn

By the end, you'll be able to —

  1. Describe the fundamental principles of human–AI collaboration and workflow automation.
  2. Identify appropriate use cases and productivity applications for automation in business, education, and service environments.
  3. Map a process and translate it into a sequence of tasks executable by agents or AI tools.
  4. Construct automation workflows using platforms like ChatGPT Agents, Zapier, Make, and Microsoft Power Automate.
  5. Apply retrieval-augmented generation (RAG) or contextual memory to improve model outputs.
  6. Assess system performance through testing protocols, error analysis, and user feedback loops.
  7. Develop ethical safeguards addressing bias, privacy, and accessibility concerns in automation design.
  8. Collaborate with peers to review, refine, and troubleshoot automated solutions.
  9. Compose professional documentation and reflective analysis summarizing workflow design, impact, and ethical considerations.
